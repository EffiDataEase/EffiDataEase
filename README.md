1. 因为需要读取密钥，所以不得不去跨进程打开句柄读取内存，只有这一步被认为是危险操作
2. 没有购买软件签名(个人开发者资金有限，需要大家多多支持，后面可以补上)，所以杀毒软件的策略会相对更严格些

### 解决方案:
1. 如果对软件信任，可以推掉杀毒软件，或者在报毒的时候请放行
2. 对于实在不放心的用户，可以在虚拟机里验证(登录的时候开启网络，后面把网络停用，确保万无一失)
